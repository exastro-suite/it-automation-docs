Exastro
IT Automation

Exastro IT Automationはシステム設定を
デジタル化して一元管理するための
オープンソースのフレームワークです。

  • Download
  • Description

Exastro IT Automation はシステム構築を取り巻く3つの問題を解決します。

  • データ多重管理による
    設計ミス
    データを一括で管理、二重管理による設計ミスを防ぎます。
  • 複雑な人手作業による
    オペレーションミス
    構築作業は自動実行、実行状況をわかりやすく表示します。
  • 作業記録不足による
    障害対応遅延
    作業記録をしっかりと管理、作業の成否をレポートします。

Exastro IT Automation 概要

Exastro IT Automation の７つの特徴です。

  1. 　マルチインタフェースとRBAC
  2. 　パラメータをグルーピング／履歴管理する
  3. 　IaCを解析して変数を刈り取る
  4. 　IaCをモジュール管理して再利用性を高める
  5. 　複数の自動化ソフトウェアを繋げて実行する
  6. 　自動化を止めない最後の切り札Pioneerモード
  7. 　実行状況をリアルタイムで監視する
